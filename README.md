# Task Planner App 📝

An advanced and user-friendly Todo List application built using **HTML, CSS, and JavaScript**.

This application helps users manage daily tasks by adding deadlines, setting priorities, and receiving reminders for important activities.

---

# Features 🚀

## Task Management

* Add new tasks
* Add tasks using **Enter key**
* Mark tasks as completed
* Delete individual tasks
* Clear all tasks

## Date & Time Management

* Set task start date
* Set task end date
* Set task start time
* Set task end time

## Priority Management

* Mark important tasks ⭐
* Mark urgent tasks 🚨
* Organize tasks based on priority

## Reminder System

* Deadline reminder alarm 🔔
* Notification for scheduled tasks

## Data Storage

* Tasks are automatically saved
* Data remains available after refreshing the page
* Uses Browser Local Storage

# How to Run the Project ▶️

### Step 1

Clone the repository:

```
git clone <your-github-repository-link>
```

### Step 2

Open the project folder in VS Code.

### Step 3

Install the Live Server extension.

### Step 4

Open:

```
index.html
```

### Step 5

Right click on `index.html`

Select:

```
Open with Live Server
```

The application will open in your browser.

---

# How to Use 📌

1. Enter your task name
2. Press Enter or click Add button
3. Add required date and time
4. Mark task as:

   * Important ⭐
   * Urgent 🚨
5. Complete tasks after finishing
6. Delete unwanted tasks
